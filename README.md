# Terraform AWS Infrastructure Project

## Overview
This project demonstrates Infrastructure as Code (IaC) using Terraform to provision scalable and reusable AWS infrastructure. It follows modular design principles and supports multiple environments (development and production).

## Architecture

The infrastructure includes:

- VPC (Virtual Private Cloud)
- Security Groups (modular)
- Multi-environment structure (dev / prod)
- Terraform state management

## Services

- Terraform
- Amazon Web Services (AWS)
- Git & GitHub
- Linux/macOS CLI

## Features

- Modular Terraform design (reusable components)
- Environment separation (dev/prod)
- Dynamic security group configuration
- Scalable cloud architecture foundation
